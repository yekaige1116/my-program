# 长征小红军数字人设计文档

## 项目概述

创建一个可交互的长征小红军数字人，采用2D卡通风格，支持AI智能对话、角色动画和长征知识展示。

## 技术方案

**方案类型**：轻量级纯前端实现
**文件形式**：单个HTML文件
**技术栈**：HTML5 + CSS3 + JavaScript + Canvas

## 整体架构

```
┌─────────────────────────────────────────────────────────┐
│                    网页交互.html                         │
├─────────────────────────────────────────────────────────┤
│  ┌─────────────┐  ┌─────────────┐  ┌─────────────┐     │
│  │  小红军角色  │  │  对话界面   │  │  功能面板   │     │
│  │  (Canvas)   │  │  (气泡/输入)│  │  (按钮/菜单)│     │
│  └─────────────┘  └─────────────┘  └─────────────┘     │
│                                                         │
│  ┌─────────────────────────────────────────────────┐   │
│  │              AI对话引擎 (国内API)                │   │
│  │         通义千问 / 文心一言 / 讯飞星火           │   │
│  └─────────────────────────────────────────────────┘   │
└─────────────────────────────────────────────────────────┘
```

## 视觉设计

### 小红军形象
- 2D卡通风格的红军小战士
- 穿着红军军装，戴八角帽，红五星
- 大眼睛，可爱亲切的造型
- 支持多个表情：开心、思考、敬礼、说话等

### 界面布局
- 左侧：小红军角色展示区（Canvas动画）
- 右侧：对话交互区（消息气泡 + 输入框）
- 底部：快捷问题按钮 + 功能切换

### 配色方案
- 主色：红军红 (#C41E3A)
- 辅色：军装灰绿 (#4A5D23)
- 背景：米黄色 (#F5F0E1) 复古纸张感

## 功能模块

### 1. 角色动画系统
- 待机动画：轻微晃动、眨眼
- 说话动画：嘴巴开合、手势
- 情绪动画：开心、思考、激动
- 动作：敬礼、走路、挥手

### 2. AI对话系统
- 支持通义千问API（默认）
- 预设长征知识库作为对话上下文
- 支持切换其他国内AI服务
- API Key 配置界面

### 3. 内容模块
- 长征历史事件介绍
- 人物故事讲解
- 路线图展示（可选扩展）
- 长征精神阐释

### 4. 交互功能
- 文字输入对话
- 快捷问题按钮
- 语音输入（可选，浏览器支持时）
- 对话历史记录

## 长征知识库内容

### 历史事件
- 飞夺泸定桥
- 四渡赤水
- 强渡大渡河
- 爬雪山过草地
- 遵义会议
- 会宁会师

### 人物故事
- 小红军的故事
- 红军将领事迹
- 普通战士的感人故事

### 地理路线
- 出发地：瑞金
- 重要途经点
- 会师地：会宁

### 长征精神
- 不怕牺牲
- 坚韧不拔
- 团结互助
- 勇往直前

## 技术实现细节

### Canvas动画
- 使用requestAnimationFrame实现流畅动画
- 精灵图或矢量绘制角色
- 状态机管理角色动作

### AI对话
- 支持多个国内AI服务商API
- 用户可配置API Key
- 预设system prompt包含长征知识

### 数据存储
- localStorage保存API Key
- localStorage保存对话历史（可选）
